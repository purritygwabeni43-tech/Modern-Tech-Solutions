# :book: Project Background

ModernTech Solutions is a software development company founded in **2010**. Over the years, the company has grown from a small start-up with only **20 employees** to a successful business employing more than **250 employees** across multiple departments, including Software Development, Quality Assurance, Customer Support, Sales, Marketing, and Human Resources.

As the company expanded, its Human Resource processes became difficult to manage. Employee information was stored across multiple Excel spreadsheets, payroll calculations were completed manually, leave requests were managed through emails, and performance reviews were saved as Word documents. This created inefficiencies, duplicated information, and increased the possibility of human error.

To solve these challenges, this project presents a **front-end Human Resource Management System** that centralizes HR activities into one easy-to-use web application. The system uses **dummy data** to simulate real HR operations and demonstrates how ModernTech Solutions could modernize its HR processes in the future.

# :rocket: Installation

Follow these steps to run the project locally.

### 1. Clone the repository

```bash
git clone github.com/purritygwabeni43-tech/Modern-Tech-Solutions
```

### 2. Navigate to the project folder

```bash

```
